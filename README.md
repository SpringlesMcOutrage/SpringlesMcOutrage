### AI-Powered CRM Analytics Platform
> Multi-tenant SaaS — sales teams connect HubSpot, Salesforce, or Pipedrive and query live pipeline data in plain English via GPT-4o.

- OAuth 2.0 + PKCE integrations for 3 CRM providers
- Auto-sync every 15 min via BullMQ/Redis job queue
- GPT-4o query layer with intent classification & self-learning memory
- Full CRM dashboard: contacts, deals, pipeline kanban, deal canvas, calendar
- Multi-tenant architecture on Supabase/PostgreSQL, deployed on Railway

**Stack:** `Python` `React` `TypeScript` `PostgreSQL` `Redis` `GPT-4o` `Railway`
